# Harshal Hand

My personal handwriting font, digitized and ready to use anywhere.

## Files

| File | Description |
|------|-------------|
| `HarshalHand-Regular.ttf` | TrueType Font (desktop install) |

## Usage

### Web (CSS @font-face)

```css
@font-face {
  font-family: 'Harshal Hand';
  src: url('https://raw.githubusercontent.com/HarshalVankudre/harshal-hand-font/main/HarshalHand-Regular.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}

body {
  font-family: 'Harshal Hand', cursive;
}
```

### Desktop Install

1. Download `HarshalHand-Regular.ttf`
2. Double-click and install
3. Select from font menu in any app

### Canva / MS Office

- **Canva**: Upload > Brand Kit > Fonts
- **Microsoft Office**: Install locally, it shows up
- **Google Docs**: Use Chrome extension
