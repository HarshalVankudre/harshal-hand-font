## Usage

### Web (CSS @font-face)

Add to your CSS to use the font on any website:

```css
@font-face {
  font-family: 'Harshal Hand';
  src: url('https://raw.githubusercontent.com/HarshalVankudre/harshal-hand-font/main/HarshalHand-Regular.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}

h1, .handwriting {
  font-family: 'Harshal Hand', cursive;
}
```

**HTML:**

```html
<h1 style="font-family: 'Harshal Hand', cursive;">Welcome to my site</h1>
```

### Desktop Install

1. **Download** `HarshalHand-Regular.ttf`
2. **Double-click** the file - click **Install**
3. **Select** from any app font menu (Word, Photoshop, etc.)
